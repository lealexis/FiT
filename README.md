# FiT (Fidelity Threshold)
Simulation and implementation of the FiT Link layer quantum communication 
protocol. A noisy communication link between Alice and Bob is established 
to enable the distribution of quantum frames from Alice to Bob. The 
quantum frame's payload is either entanglement a.k.a. 
EPR(Einstein-Podolsky-Rosen)-Pairs or superdense coded information i.e.
EPR-frame or SDC-frame. 


The distributed entanglement satisfies a 
minimal threshold fidelity.
